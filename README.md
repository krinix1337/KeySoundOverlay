**KeySound Overlay** — это быстрое, минималистичное и настраиваемое Windows-приложение, созданное в стиле **Fluent Windows 11**. Приложение воспроизводит реалистичные звуки нажатий механических клавиатур при каждом физическом клике, а также отображает полупрозрачную интерактивную оверлей-панель клавиатуры прямо поверх всех окон.

> [!IMPORTANT]
> Приложение использует низкоуровневый системный хук исключительно для подсветки клавиш оверлея и воспроизведения звуков. Введенный текст нигде не сохраняется, не логируется и не передается по сети.

## ✨ Ключевые Особенности

*   **⚡ Сверхнизкая задержка звука:** Инициализация аудиомотора `pygame.mixer` с аппаратной буферизацией в 512 байт гарантирует мгновенный отклик звука даже при скорости печати более 500 символов в минуту.
*   **🎹 Живой органический звук (WAV Pitch-Shift):** Для каждого свитча генерируется пул из 5 тембральных вариаций. При нажатии клавиши высота тона и громкость динамически варьируются на ±10%, создавая реалистичное звучание настоящей механической клавиатуры.
*   **📐 Динамическое масштабирование и позиционирование:** Оверлей больше не занимает весь экран! Размер окна автоматически подстраивается под выбранный игровой режим:
    *   `Полная клавиатура` — аккуратная ANSI-раскладка (`610x215px`).
    *   `WASD Игровой` — компактный блок клавиш WASD + Shift + Space + Ctrl (`270x140px`).
    *   `Osu! Кликер` — специализированный блок Z + X + Space + Esc (`150x140px`).
    *   `Dota 2 Панель` — скиллы, предметы, цифры и модификаторы (`340x170px`).
    *   `Своя раскладка (Custom)` — динамический размер по списку ваших клавиш.
    *   `История нажатых клавиш (Pressed)` — горизонтальная лента логов (`360x50px`).
*   **🔒 WinAPI Click-Through:** В заблокированном состоянии оверлей на 100% игнорирует клики мыши, пропуская их в окна под ним. В разблокированном — оверлей обзаводится стильной рамкой и свободно перетаскивается в любое место экрана.
*   **🛡️ Двойная защита от копий:** Интегрированная система предотвращения запуска нескольких инстансов через `QLockFile` и `QSharedMemory` гарантирует, что на экране всегда будет только один оверлей и одна иконка в трее.
*   **📦 Профессиональный инсталлятор (без UAC и %TEMP% ошибок):** Новая сборка через Inno Setup устанавливает приложение локально в один клик, не вызывая ложных срабатываний антивирусов и ошибок доступа.

## 🎨 Темы Оформления

Вы можете выбрать одну из 4 встроенных тем оформления во вкладке **Внешний вид**:
1.  **Dark (Темная Fluent)** — классическая приглушенная темная тема Windows 11.
2.  **Light (Светлая Fluent)** — аккуратная бело-серая тема с хорошей контрастностью.
3.  **Glass (Полупрозрачное стекло)** — эффект матового стекла с тонкими рамками.
4.  **Neon Blue (Киберпанк)** — футуристичная темная тема с неоновой голубой подсветкой.

## 🚀 Установка и Сборка

> [!TIP]
> Для самой простой установки используйте готовый инсталлятор, не требующий наличия Python на компьютере.

### Вариант 1: Быстрая установка (Рекомендуется)
1.  Перейдите в раздел **Releases** на главной странице этого репозитория на GitHub (вкладка справа).
2.  Скачайте готовый установщик **`KeySoundOverlay_Setup.exe`** последней версии.
3.  Запустите скачанный файл и следуйте простым инструкциям. Он автоматически установит приложение, создаст ярлыки на Рабочем столе и пропишет его в меню Пуск.

### Вариант 2: Установка из исходного кода
1.  Установите библиотеки:
    ```bash
    pip install -r requirements.txt
    ```
2.  Запустите приложение:
    ```bash
    python main.py
    ```

### Вариант 3: Ручная установка через скрипт
Запустите файл **`install.bat`** в корне папки. Он вызовет PowerShell-скрипт установки, который скопирует приложение, создаст ярлыки и настроит систему удаления.

## 🛠️ Компиляция проекта (Для разработчиков)

> [!TIP]
> Подробные пошаговые инструкции по созданию своих раскладок клавиатуры, интеграции новых звуков свитчей, кастомных тем и публикации релизов на GitHub находятся в документе [DEVELOPER_GUIDE.md](DEVELOPER_GUIDE.md).

Если вы внесли изменения в код и хотите перекомпилировать приложение:

1.  Убедитесь, что у вас установлен PyInstaller (`pip install pyinstaller`).
2.  Для сборки основной папки приложения выполните:
    ```bash
    pyinstaller --noconfirm KeySoundOverlay.spec
    ```
3.  Для сборки инсталлятора запустите скрипт компиляции Inno Setup:
    *   Убедитесь, что Inno Setup установлен на компьютере.
    *   Запустите компиляцию скрипта `setup.iss` через компилятор Inno Setup.

## 📋 Системная информация

*   **Файл настроек:** `%APPDATA%/KeySoundOverlay/settings.json`
*   **Автозагрузка:** Записывается в ветку реестра `HKCU:\Software\Microsoft\Windows\CurrentVersion\Run\KeySoundOverlay`
*   **Удаление:** Нативно интегрировано в раздел Windows «Установка и удаление программ».
