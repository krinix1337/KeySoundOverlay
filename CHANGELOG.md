# Changelog — KeySound Overlay

Все значимые изменения по версиям.

---

## [1.3.0] — 2026-06-24

### ✨ Новое

- **7 новых звуковых пресетов** — добавлены синтетические звуки реальных свичей:
  - 🟡 Gateron Yellow — ультра-тихий гладкий линейный
  - 🟠 Alps SKCM Orange — винтажный тактильный с двойным ударом
  - ⬜ Kailh Box White — чёткий box-клик с резонансом корпуса
  - ⚡ Speed Silver — сверхкороткий быстрый линейный
  - 🔵 Topre 45g — глубокий низкочастотный thock
  - 🔇 Boba U4 Silent — беззвучный мягкий тактильный
  - 🩷 Akko CS Jelly Pink — гладкий тактильный с лёгкими гармониками

- **Профили настроек** — теперь можно сохранять и загружать именные наборы настроек.  
  Удобно для переключения между профилями «Игра», «Стрим», «Работа»  
  *(Настройки → Профили)*

- **Ripple-анимация клавиш** — при нажатии клавиши в оверлее расходится белый световой круг (~60 fps).  
  Можно отключить в *Настройки → Оверлей → Эффекты*

- **Оверлей поверх полноэкранных приложений** — новая опция позволяет не скрывать оверлей при переходе в fullscreen-игры (работает для borderless fullscreen)  
  *(Настройки → Оверлей → Эффекты)*

- **Новый дизайн окна настроек** — вместо вкладок теперь боковая панель навигации.  
  Разделы: Звук / Оверлей / Внешний вид / Профили / Система

- **Проверка обновлений при запуске** — можно отключить в *Настройки → Система*

### 🐛 Исправления

- Исправлен адрес GitHub-репозитория для OTA-обновлений — ранее приложение проверяло несуществующий репозиторий и всегда молчало
- Устранено ограничение минимальной прозрачности оверлея в 10% — теперь можно выставить 0%

### ⚙️ Изменения

- Версия приложения: `1.2.0` → `1.3.0`
- Добавлены конфигурационные ключи: `show_in_fullscreen`, `check_updates_on_startup`, `key_press_animation`

## [1.2.0] — 2026-06-20

### ✨ Новое

- **Умный лог клавиш** — лог в оверлее учитывает активную системную раскладку: если включён русский язык (Shift+Alt), в логе пишутся кириллические символы, если английский — латиница
- **Комбинации клавиш** — оверлей корректно отображает системные сочетания: Win+S, Ctrl+Z, Alt+Tab и другие
- Режим оверлея «История нажатых клавиш (Pressed)»

### 🐛 Исправления

- Исправлена неровная иконка курсора мыши в оверлее
- Улучшено выравнивание элементов интерфейса настроек

---

## [1.1.0] — 2026-06-19

### ✨ Новое

- **11 пресетов механических свичей** из паков Mechvibes (Cherry MX Blue, Brown, Red, Black, Holy Panda, Turquoise Tealio, NK Cream, Cream Travel, EG Oreo, Crystal Purple, Topre)
- **Живой звук** — pitch-shift ±10% и вариация громкости для органичного звучания
- **Оверлей мыши** — визуализация кнопок (LMB/RMB/MMB/X1/X2), координат и колёсика
- **4 темы** — Dark, Light, Glass, Neon Blue
- **Конструктор кастомных тем** в настройках
- **6 режимов оверлея** — Full, WASD, Osu!, Dota2, Custom, Pressed
- Установщик на базе Inno Setup (без прав администратора)
- Портативная версия (ZIP)
- OTA-обновления через GitHub Releases
- Автозапуск с Windows
