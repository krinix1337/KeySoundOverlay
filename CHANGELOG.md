# Changelog

## [1.4.0] - 2026-06-24

### ✨ Что нового
- **Новые звуки свитчей**: 7 новых звуковых паков были заменены на реальные качественные аудио-сэмплы (Gateron Yellow, Alps SKCM, Kailh Box White, Speed Silver, Topre 45g, Boba U4, Akko Jelly Pink). 
- **Новые векторные иконки**: Все иконки интерфейса были заменены на красивые профессиональные (Lucide Icons) для улучшения эстетики.
- **Плавные анимации интерфейса**: Добавлены анимации затухания (fade-in) при переключении между вкладками настроек.
- **Магнитное выравнивание (Snap)**: При перетаскивании оверлеев с клавиатурой или мышкой они автоматически примагничиваются к центру экрана (по вертикали и горизонтали), что позволяет идеально ровно их позиционировать!
- **Улучшенный инсталлятор**: В программу установки добавлены красивые неоновые баннеры и описание программы.

### 🐛 Исправления (Багфиксы)
- Исправлена ошибка сохранения и воспроизведения анимаций клавиш (теперь в настройках доступен удобный выпадающий список выбора типа анимации).
- Исправлено отображение тем (Dark, Glass, Neon) для нового бокового меню навигации. Боковое меню теперь корректно меняет стили в зависимости от выбранной темы.
- Оптимизировано удаление анимаций нажатий из памяти (таймер теперь корректно останавливается).
