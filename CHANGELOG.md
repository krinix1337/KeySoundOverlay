# Changelog

## [1.5.2] - 2026-06-25
### 🐛 Исправления
- Добавлена кнопка сворачивания в окне настроек.
- Переключатели теперь реагируют на нажатие по всему блоку с текстом.
- Разблокировка оверлея объединена в одну кнопку (и для клавиатуры, и для мыши).
- Окно обновления теперь принимает цвета установленной темы приложения.
- Исправлено вылезание анимации клавиш за скругленные границы.
- Добавлен глобальный фон для всех окон приложения, теперь темы применяются абсолютно ко всем элементам интерфейса.


## [1.5.2] - 2026-06-25
### 🐛 Исправления (Багфиксы)
- **Глобальное применение тем**: Исправлена ошибка, из-за которой новые стили (переключатели, карточки, сайдбар) не перекрашивались при смене тем (Dark, Light, Glass, Neon). Теперь темы работают на 100% корректно.


## [1.5.2] - 2026-06-25

### ✨ Что нового
- **Редизайн настроек**: Полностью переработан интерфейс настроек. Убран визуальный мусор, заменены стандартные галочки на красивые iOS-подобные переключатели (`ToggleSwitch`), интерфейс стал чище и удобнее.
- **Магнитные линии прицеливания (Snap Guides)**: При перетаскивании оверлеев теперь отображаются 6 красивых пунктирных направляющих линий на всех мониторах. Оверлеи автоматически примагничиваются к краям и центру экрана для идеально ровного позиционирования.

### 🐛 Исправления (Багфиксы)
- **Исправлен чёрный экран на Windows 10**: Заменены прозрачные фоны `rgba()` на плотные цвета в теме Glass, которые вызывали черный экран на старых версиях Windows 10. Удалены несуществующие шрифты `Segoe UI Variable Text`.
- Исправлена логика ложного срабатывания "Доступно обновление", когда уже установлена актуальная версия.
- Исправлено применение стилей к боковому меню при смене тем.
- Мелкие исправления в механике работы оверлея мыши.

## [1.4.0] - 2026-06-24

### ✨ Что нового
- **Новые звуки свитчей**: 7 новых звуковых паков были заменены на реальные качественные аудио-сэмплы (Gateron Yellow, Alps SKCM, Kailh Box White, Speed Silver, Topre 45g, Boba U4, Akko Jelly Pink). 
- **Новые векторные иконки**: Все иконки интерфейса были заменены на красивые профессиональные (Lucide Icons) для улучшения эстетики.
- **Плавные анимации интерфейса**: Добавлены анимации затухания (fade-in) при переключении между вкладками настроек.
- **Улучшенный инсталлятор**: В программу установки добавлены красивые неоновые баннеры и описание программы.

### 🐛 Исправления (Багфиксы)
- Исправлена ошибка сохранения и воспроизведения анимаций клавиш (теперь в настройках доступен удобный выпадающий список выбора типа анимации).
- Оптимизировано удаление анимаций нажатий из памяти (таймер теперь корректно останавливается).
